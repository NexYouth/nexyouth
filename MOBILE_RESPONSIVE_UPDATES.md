# 移动端适配更新说明

## 概述
为 NexYouth 网站添加了完整的移动端响应式设计支持，确保在手机和平板设备上有良好的用户体验。

## 主要更改

### 1. 导航菜单优化
- **添加了汉堡菜单按钮** - 在屏幕宽度 ≤768px 时显示
- **响应式导航菜单** - 手机上以垂直菜单形式展开
- **移动下拉菜单处理** - Programs 菜单在移动设备上支持点击展开
- **点击外部自动关闭** - 点击导航外部时自动关闭菜单

### 2. CSS 媒体查询更新
为所有页面添加了两个断点的响应式设计：
- **@media (max-width: 768px)** - 平板和较小屏幕
- **@media (max-width: 480px)** - 手机屏幕

#### 响应式调整包括：
- **导航栏**
  - Logo 高度缩小 (70px → 50px → 40px)
  - 导航菜单在小屏幕上隐藏并改为垂直布局
  - 添加了汉堡菜单按钮样式

- **Hero 部分**
  - 标题字体大小缩放 (3.5rem → 1.8rem → 1.4rem)
  - 副标题字体大小调整 (1.3rem → 1rem → 0.95rem)
  - 按钮大小和间距优化

- **网格布局**
  - 4列网格改为1列 (程序卡片、团队成员等)
  - 2列网格改为1列 (统计数据在最小屏幕)
  - 间距和外边距调整

- **文字和内容**
  - 所有主标题 (h2) 字体大小缩小
  - 正文字体大小适应屏幕
  - 内边距和外边距在小屏幕上减少

- **表单和输入**
  - 表单宽度100%填充
  - 标签和输入框优化尺寸
  - 改善触摸友好性

### 3. JavaScript 交互功能
在所有 HTML 文件中添加了移动菜单的 JavaScript：
```javascript
- 汉堡菜单按钮点击处理
- 导航链接点击时自动关闭菜单
- 下拉菜单移动端适配
- 点击外部关闭菜单
```

## 新增 CSS 类

### .mobile-menu-btn
- 汉堡菜单按钮样式
- 只在 ≤768px 屏幕显示
- 点击时有动画效果 (X 形变换)

### nav ul 媒体查询样式
- 小屏幕上隐藏
- `.active` 类时显示为垂直菜单
- 全屏宽度下拉菜单

## JavaScript 功能

### 菜单控制
1. **mobileMenuBtn.click** - 切换菜单显示/隐藏
2. **navLinks.click** - 点击导航链接时关闭菜单
3. **dropdownTrigger.click** - 移动设备上处理下拉菜单
4. **document.click** - 点击外部时关闭菜单
